TOPIC
    about_LoopbackAdapter

SHORT DESCRIPTION
    PowerShell module for working with Loopback Adapters.

LONG DESCRIPTION
    This module provides support for creating and removing Loopback Network Adapters
    on Windows using Device Console (DevCon.exe).

EXAMPLES
    PS C:\> Get-Command -Module LoopbackAdapter
